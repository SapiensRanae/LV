### Prerequisites

- [Node.js](https://nodejs.org/) & npm

### Frontend Setup

```bash
cd ClientSideFE
npm install
npm start
```

The app will be available at `http://localhost:8080`.

### Backend Setup

```bash
cd ServerSide
dotnet build
dotnet run
```

The API will be available at `http://localhost:5151` (or as configured).
